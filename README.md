# CHUNITHM Selection Display (ver 0.2-alpha)

## 描述
它可以在中二节奏比赛中展示玩家的选曲，也可以随机选择一首乐曲并展示。

## 已实现功能：
- 在全曲曲库中选歌（全曲曲库指截止2025/12/27的国服曲库）。
- 在全曲库中随机一首歌曲。

## 使用说明：
- 指定选曲时需输入乐曲的对应ID。

---

## Description
It can display the players' music selections in a CHUNITHM match. It can also pick a music at random and display it.

## Features Implemented:
- Song selection from the full music library (as of December 27, 2025, based on the Chinese server's catalog).
- Random song selection from the entire library.

## Instructions:
- When specifying a music to select, you must input the unique music ID associated with that music.
